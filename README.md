# apple-test

Минимальный пример TypeScript/Node.js проекта с автосборкой через GitHub Actions.

## Запуск

```bash
npm install
npm run build
npm start
```

## Сборка вручную через GitHub Actions

Перейдите во вкладку Actions на GitHub и запустите workflow вручную, если это предусмотрено (например, через кнопку Run workflow).
